<h1 align="center"> BlockControl </h1>

## 🇺🇸 English

This project is an interactive command-line interface (CLI) designed to simplify the management of blocked IP addresses, domains, and email addresses in Microsoft Exchange. Instead of typing complex syntax or script parameters manually, administrators can navigate through a friendly menu using the keyboard arrow keys to easily block or unblock entries and view the current list of blocked items.

The tool was developed entirely with PowerShell, using only native .NET Framework console controls on Windows, and focuses solely on simplifying these operations without maintaining external logs or databases.

Its goal is to make the management of blocked entries more practical and accessible, avoiding the need to manually execute long PowerShell commands or navigate complex administrative menus in the Exchange environment.

## 🇧🇷 Português

Este projeto é uma interface de linha de comando (CLI) interativa desenvolvida para simplificar o gerenciamento de IPs, domínios e endereços de e-mail bloqueados no Microsoft Exchange. Em vez de digitar sintaxes complexas ou parâmetros manualmente, o administrador pode navegar por um menu amigável usando as setas do teclado para bloquear ou desbloquear facilmente os itens, além de visualizar a lista atual de bloqueios.

A ferramenta foi desenvolvida inteiramente em PowerShell, utilizando apenas os controles de console nativos da .NET Framework no Windows, e tem como foco apenas facilitar essas operações — sem registrar logs, nem atualizar bases de dados externas.

O objetivo é tornar a administração de bloqueios mais prática e acessível, evitando a necessidade de uso manual de comandos longos no PowerShell ou a navegação por menus administrativos complexos do ambiente Exchange.
